# UI Targets

Short labels for quick UI references.

- Topbar: `header.topbar` (actions row)
- Canvas panel: `.canvas-panel` -> `#canvas` -> `.page-surface`
- Sidebar: `aside.sidebar`
- Hierarchy section: `#addPage`, `#treeList` (pages/containers/buttons)
- Element section: `h2` "Element" + `#elementsList`
- Status line: `#status`
